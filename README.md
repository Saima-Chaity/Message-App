# Message-App

This is a simple message application which allows authenticated users to send text messages and images.

### Technology Stack:
- Angular

- Firebase

- Bootstrap

- Azure Devops

### Features:

- Login / Signup 

- User profile settings

- Send simple text messages and images

- Available user list
